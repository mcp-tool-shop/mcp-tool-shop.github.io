How It Works

# Before: 77 tools x 500 tokens = 38,500 tokens per request
# After: 1 compass tool + 3 results = 2,000 tokens per request

# Step 1: Find tools by intent
compass("generate an AI image from text")
# Returns: comfy:comfy_generate (0.912 confidence)

# Step 2: Get full schema
describe("comfy:comfy_generate")
# Returns: Complete parameter schema

# Step 3: Execute
execute("comfy:comfy_generate", {
    "prompt": "a beautiful sunset"
})

MCP Tools

compass(intent) Semantic search for tools matching your intent
describe(tool_name) Get full schema for a specific tool
execute(tool, args) Run a tool on its backend server
compass_categories() List available categories and servers
compass_status() System health and configuration
compass_analytics() Usage statistics and performance metrics
compass_chains() Manage tool workflows and sequences
compass_sync() Rebuild index from backend servers
