Claude-Fresh

Automatic context checkpoints for Claude Code. No forced resets. You decide when to /compact.

Research-backed: Lost in the Middle (Stanford 2023), Context Rot (Chroma 2025)
pip install claude-fresh

Requires Python 3.11+ and Claude Code

Automatic Handoffs

At 75% context, Claude writes a comprehensive markdown handoff. No intervention required.

Searchable Index

SQLite-backed full-text search by keyword, file, or decision. Find any past context instantly.

Group Summaries

Every 10 handoffs, Claude-Fresh creates a group summary. Search at two levels: details or patterns.

Verbatim Messages

Last 5 messages preserved exactly. No lossy summarization at the critical handoff point.

Human-Readable

Markdown files you can browse in any editor. Version-controllable. Grep-able.

User Controls Reset

Checkpoint saved, but you decide when to /compact. Critical moment? Stay in Phase 2.

MCP Tools

fresh_status Current context % and handoff count
fresh_save Create a handoff (Claude writes the content)
fresh_search Search handoffs by keyword, file, or decision
fresh_load Load a specific handoff into context
fresh_latest Load the most recent handoff
fresh_groups List group summaries
