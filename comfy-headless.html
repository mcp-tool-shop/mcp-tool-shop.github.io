Comfy Headless

Making ComfyUI's power accessible without the complexity. AI image & video generation in 3 lines of code.

v2.5.1 | LTX-Video 2, Hunyuan 1.5, Wan 2.1/2.2 support
pip install comfy-headless[standard]

Requires ComfyUI running locally (default: localhost:8188)

Quick Start

from comfy_headless import ComfyClient

client = ComfyClient()
result = client.generate_image("a beautiful sunset over mountains")
print(f"Generated: {result['images']}")

# With AI prompt enhancement
from comfy_headless import enhance_prompt
enhanced = enhance_prompt("a cat", style="detailed")
# "a cat, masterpiece, best quality, highly detailed..."

# Video generation
result = client.generate_video(
    prompt="a cat walking through a garden",
    preset="ltx_quality"  # LTX-Video 2, 1280x720, 49 frames
)
