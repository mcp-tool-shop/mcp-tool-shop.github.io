How Resolution Works

1 tool@channel ‚Üí Registry Pointer (signed)
2 Pointer ‚Üí descriptor_cid + root_cid
3 descriptor_cid ‚Üí tool.descriptor.json (metadata, policy)
4 root_cid ‚Üí toolbundle.manifest.json (file list)
5 Attestations ‚Üí Verify signatures, check expiry, enforce constraints
6 Policy Gate ‚Üí ACCEPT or REJECT with error code

Key Error Codes

Code Description
MANIFEST_CID_MISMATCH Computed CID doesn't match manifest.root_cid
DESCRIPTOR_CID_MISMATCH Computed descriptor CID doesn't match pointer
NO_VALID_ATTESTATIONS No attestations found for this artifact
ATTESTATION_EXPIRED Integrity claim past expiry date
POLICY_BLOCKED_NETWORK Tool requires network access, installer denies

See Rulebook §7 for the complete error code reference.
