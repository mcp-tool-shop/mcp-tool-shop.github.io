Quick Start

# Index your code
file-compass index -d "C:/Projects"

# Search semantically
file-compass search "database connection handling"

# Filter by file type
file-compass search "training loop" --types python

# Git-tracked files only
file-compass search "API endpoints" --git-only

# Quick search (no embeddings)
file-compass scan -d "C:/Projects"

Semantic Search

Find files by describing what you're looking for. "database connection" finds connection.py, db_handler.ts, etc.

Quick Search

Instant filename and symbol search without embeddings. Sub-10ms for common queries.

Multi-Language AST

Tree-sitter support for Python, JavaScript, TypeScript, Rust, Go. Function and class-level chunking.

Result Explanations

Understand why each result matched. Not just a score, but a reason.

Local Embeddings

Uses Ollama with nomic-embed-text. No API keys needed. Runs entirely on your machine.

Security Hardened

Input validation, path traversal protection, SQL injection prevention. Errors sanitized.

How It Works

1

Scanning

Discovers files matching extensions, respects .gitignore

2

Chunking

AST-aware via tree-sitter (functions, classes)

3

Embedding

768-dim vectors via Ollama

4

Indexing

HNSW index + SQLite metadata

5

Search

Embed query, find nearest neighbors
